Admin Access Roles

Admin users have different levels of access to the Glynac analytics system. Assign roles based on responsibilities.

System Admin

Full access to all features, including user management, data sources, and system settings.

Department Admin

Access limited to specific departments. Can view analytics and manage employees within assigned departments.

Analytics User

View-only access to analytics dashboards. Cannot manage employees or system settings.
